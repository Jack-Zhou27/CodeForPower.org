PRELIMINARY: Getting Started

A Brief Description of Java:
Java is a popular computer programming language that is used by over 10 million people and 8 billion devices.
It is useful for developing websites, apps, games, applications, and more.
Java is also the only language taught in AP classes, and is emphasized in college courses & competitive programming (USACO, Google Code Jam etc.).
Since this language is closely related to other programming languages such as C++ and C#,
understanding Java can make you gain knowledge that transcends the limited boundary of a single language.
